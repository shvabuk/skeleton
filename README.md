# Skeleton

Skeleton with basic gulp settings (js, scss, php twig)

## Main file

dist/index.html

## Install packages

```shell
npm i -g gulp
npm i
npm up
composer install
composer update
```

## Run watcher [http://localhost:3000/](http://localhost:3000/)

```shell
gulp dev
```
